☁ Sync Across Devices
One-time setup (about 5 minutes):
  1. Go to console.firebase.google.com → Create project
  2. Build → Firestore Database → Create database (production mode)
  3. Build → Authentication → Get started → enable Email/Password
  4. Project settings → Your apps → Web </> → register app → copy the config
  5. Paste keys into firebase-config.js in the project and set enabled: true
  6. Firestore → Rules → paste rules from firestore.rules → Publish
  7. Refresh this app, then create your sync account below
Sign in with the same email and password on your phone and PC. Changes sync automatically — no importing.
